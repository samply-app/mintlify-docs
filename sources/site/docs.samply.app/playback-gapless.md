# Gapless Playback [​](https://docs.samply.app/#gapless-playback)

When listening in Samply audio plays back gaplessly, meaning that adjacent tracks in the sequence are stitched together into one contiguous audio stream. This is essential for previewing projects that have cross-fades or other seamless transitions between tracks.

### Exceptions [​](https://docs.samply.app/#exceptions)

Since Samply is a collaborative platform, the app reacts to any changes in the track-list in realtime. This, however, can be a challenge if audio is added to a project while it is being played. Currently, gapless transitions are temporarily disabled while the track list is changing.

## iOS App [​](https://docs.samply.app/#ios-app)

There are virtually no exceptions to playback in the native [iOS App](https://docs.samply.app/./applications.html#ios). This environment is always gapless and is generally more robust than browser environments.
