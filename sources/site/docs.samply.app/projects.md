# Projects [​](https://docs.samply.app/#projects)

All files in Samply are stored within projects, each with its own [file structure](https://docs.samply.app/./file-system.html) and [user access settings](https://docs.samply.app/./sharing.html#collaborators). Understanding projects helps you optimize your workflow.

![Project View](https://docs.samply.app/img/projects/project.gif)

## View Modes [​](https://docs.samply.app/#view-modes)

Projects have two view modes: [Listen](https://docs.samply.app/#listen-mode) & [Manage](https://docs.samply.app/#manage-mode), each suited for different tasks.

Switch between them via the **Toolbar** or by pressing `\` on your keyboard.

![View Mode Toggle Button](https://docs.samply.app/img/projects/view-mode-toggle.png)

### Listen Mode [​](https://docs.samply.app/#listen-mode)

A clean, streamlined interface for listening—this is what listeners see when you share a [Public Link](https://docs.samply.app/./sharing.html#public-links).

### Manage Mode [​](https://docs.samply.app/#manage-mode)

A detailed, data-rich view for organizing files, with support for [customizable columns](https://docs.samply.app/#data-columns).

## Data Columns [​](https://docs.samply.app/#data-columns)

Customize column visibility based on what matters most. Tap the **3-dot menu** next to the column headers or right-click to adjust settings.

![Data Column Editing](https://docs.samply.app/img/projects/data-columns.gif)

## Billing & Ownership [​](https://docs.samply.app/#billing-ownership)

- The **project owner** is the user who created the project.
- Storage usage counts toward the owner's quota.
- Ownership can be transferred to a collaborator via the **Share Dialog** → "Transfer ownership." The new owner gains full control once accepted.

## Deleting a Project [​](https://docs.samply.app/#deleting-a-project)

- Right-click a project in the **Library** and select **Delete**.
- Enable **Auto-Archive** in the project settings to move inactive projects to archives after a set time.
