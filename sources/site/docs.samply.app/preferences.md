# Preferences [​](https://docs.samply.app/#preferences)

Customize your Samply experience with settings for notifications, audio, project defaults, and advanced features.

![Preferences View](https://docs.samply.app/img/preferences/preferences.png)

## Notifications [​](https://docs.samply.app/#notifications)

Control when and where you receive notifications.

- **Push Notifications** – Enable browser-based notifications for real-time updates.
- **Email Digests** – Get an activity summary on your schedule (e.g., once a day).

## Audio [​](https://docs.samply.app/#audio)

Adjust playback quality and loudness normalization.

> _Note:_ You must be playing a track to see available audio options.

## Project Defaults [​](https://docs.samply.app/#project-defaults)

Set default preferences for new projects.

- **Artwork** – Upload a default image for your projects.
- **Sort Order** – Choose the default file sorting method (e.g., Custom).
- **Auto Delete** – Automatically delete projects after a period of inactivity (e.g., Never).

## Advanced [​](https://docs.samply.app/#advanced)

Additional customization options for your experience.

- **Homepage** – Set the default view when launching Samply (e.g., Projects).
- **Drag to Stack** – Enable stacking by dragging a file onto another file.
- **Auto Stack** – Automatically stack new files with similar names.
- **Consolidate Players** – Display only one player per project in the library.
- **Intro Animations** – Show an animation when players are first opened.
- **Enable Messaging Sounds** – Play a sound when receiving in-app messages.
- **Formatted Links** – Copy links formatted using your project title.

## Beta [​](https://docs.samply.app/#beta)

Experimental features that may be unstable.

- **New Interface** – Unifies projects & players into a single concept.
- **Auto Rename** – Automatically renames stack titles while preserving the original filename (restart required).
- **Auto Favorite** – Automatically favorites new projects for quick access.
- **Sync Links** – Share a link that lets multiple users listen to tracks simultaneously.
- **Dynamic Backgrounds** – Enable a new animation style for intros and project backgrounds.
- **Custom Animations** – Create custom intro animations using Rive.app.
- **Samply Next** – Access the latest builds (restart required, may be unstable).
