# Interface Overview [​](https://docs.samply.app/#interface-overview)

Samply's interface is built to be efficient for power-users while remaining accessible to newcomers. There are a few common elements to the interface which are described below.

## Toolbar [​](https://docs.samply.app/#toolbar)

The toolbar is located at the top of the application. It's content changes depending on what view you are in.

![The Toolbar](https://docs.samply.app/img/overview/toolbar.png)

## Navigation Panel [​](https://docs.samply.app/#navigation-panel)

The **Navigation Panel** is located on the left side of the application. If you don't see it, click on the icon at the top left to open it or use the `[` keyboard shortcut.

![The Navigation Panel](https://docs.samply.app/img/overview/navigation-panel.png)

## Options Panel [​](https://docs.samply.app/#options-panel)

The **Options Panel** is found on the right side of any project that you have edit access on. If you don't see it, click on the icon at the top right to open it or use the `]` keyboard shortcut.

From this panel you can edit all options for a given project inluding artwork, color, access and more.

![The Options Panel](https://docs.samply.app/img/overview/options-panel.png)
