The Samply REST (Representational State Transfer) API enables you to programmatically interact with the core primitives that power your Samply workflow: 
projects, players, files and comments.

<Info>
  The Samply API is currently available on [paid plans](https://samply.app/pricing) and upon request. The Samply team can work with you to build custom solutions and extend the API wherever you need it.
</Info>

## Base URL

All API requests are made to the following base URL:

```
https://samply.app/api/v0
```
