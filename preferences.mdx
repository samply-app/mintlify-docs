Samply's Preferences page gives you control over every aspect of your experience — from how you're notified about activity to how your projects behave by default. Open Preferences from the sidebar to find settings grouped into four areas: Notifications, Audio, Project Defaults, and Advanced.

## Notifications

Choose how and when Samply keeps you informed:

- **Push Notifications** — Enable browser-based notifications for real-time alerts as activity happens.
- **Email Digests** — Receive a scheduled summary of activity across your projects (for example, once a day).

## Audio

Adjust playback quality and loudness normalization to match your listening environment.

<Note>
  You must have a track actively playing to see all available audio options. Start playback first, then open Preferences to access the full set of controls.
</Note>

## Project Defaults

Set baseline preferences that apply to all new projects you create:

- **Artwork** — Upload a default image to use as cover art for new projects.
- **Sort Order** — Choose how files are sorted by default (for example, Custom order).
- **Auto Delete** — Automatically delete projects after a period of inactivity. Set to **Never** to keep projects indefinitely.

## Advanced

Fine-tune your Samply workflow with these additional options:

- **Homepage** — Choose the default view when you open Samply (for example, Projects).
- **Drag to Stack** — Stack files by dragging one file onto another.
- **Auto Stack** — Automatically group newly uploaded files with other files that share similar names.
- **Consolidate Players** — Show only one player per project in your library view.
- **Intro Animations** — Display an animation when a player opens for the first time.
- **Enable Messaging Sounds** — Play an audio cue when you receive an in-app message.
- **Formatted Links** — When copying a project link, include the project title as formatted text.

## Beta

<Note>
  Beta features are experimental and may be unstable. Enable them to try the latest functionality, but be aware that behaviour may change or features may be removed in future updates.
</Note>

- **New Interface** — Unifies projects and players into a single concept.
- **Auto Rename** — Automatically renames stack titles while preserving the original filename. Requires a restart to take effect.
- **Auto Favorite** — Automatically favorites newly created projects for quick access.
- **Sync Links** — Generate a link that lets multiple listeners play tracks simultaneously in real time.
- **Dynamic Backgrounds** — Enables a new animation style for intro screens and project backgrounds.
- **Custom Animations** — Create and use custom intro animations built with [Rive.app](https://rive.app).
- **Samply Next** — Opt in to the latest Samply builds before they are broadly released. Requires a restart and may be unstable.
