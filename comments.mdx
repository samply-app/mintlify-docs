Comments let you capture ideas, give feedback, and keep notes directly on a track — right at the exact moment in the audio where they belong. You can leave time-coded comments pinned to a specific timestamp or general comments that apply to the track as a whole.

## Open the Comment View

The comment view is also called the **File Preview**. How you open it depends on whether the track is currently playing.

<Steps>
  <Step title="Open the File Preview for a playing track">
    If the track you want to comment on is already playing, click its title in the **Now Playing** bar at the bottom of the screen. The File Preview opens immediately.
  </Step>
  <Step title="Open the File Preview from your project files">
    If the track isn't playing, navigate to your project files and click directly on the file. If the file already has comments, you can also click the **wafer** indicator (the comment count chip) to jump straight to the comment view.
  </Step>
  <Step title="Open the File Preview from a player">
    From any shared player, click the track title or the comment indicator to open the File Preview in the same way.
  </Step>
</Steps>

## Leave a Comment

Once the File Preview is open, you have two types of comments to choose from.

<Tabs>
  <Tab title="Time-coded Comment">
    Pause or play the track to the moment you want to reference, then type your comment in the input field. The comment is stamped with the current playback timestamp and appears as a marker on the waveform. Listeners can click the marker to jump directly to that moment.
  </Tab>
  <Tab title="General Comment">
    Type your comment in the input field without stopping at a specific moment. General comments are not attached to any timestamp and appear in the comment list without a time marker.
  </Tab>
</Tabs>

## Format Your Comment

Make your feedback easier to read by applying basic formatting while you type.

| Format | Shortcut |
|--------|----------|
| **Bold** | `⌘B` |
| *Italic* | `⌘I` |

## Download All Comments

You can export every comment on a track as a plain text file for sharing, archiving, or review outside of Samply.

<Steps>
  <Step title="Open the command palette">
    Press `⌘K` to open the command palette from anywhere in the app.
  </Step>
  <Step title="Select Download comments">
    Type or select **Download comments** from the list of available actions.
  </Step>
  <Step title="Find your file">
    The text file appears in your system downloads folder shortly after.
  </Step>
</Steps>

<Tip>
  Export comments before a review meeting to give collaborators a printed or shareable summary of all feedback on a project.
</Tip>
